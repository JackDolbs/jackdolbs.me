# HTML Cheatsheet

[[toc]]

## Structure

```html
<html></html> <!-- Wraps the entire webpage  -->
<head></html> <!-- For your webpage metadata  -->
<body></html> <!-- Wraps everything that will be visible on the website  -->
<nav></html> <!-- A wrapper for your navigation  -->
<section></section> <!-- Sections  -->
<div></div> <!-- Tag from grouping elements -->
<footer></footer> <!-- A wrapper for your footer info  -->
```

Typical template:

```html
<html>
    <head></head>
    <body>
        <nav></nav>
        <section>
            <div>Some content<div>
            <div>Some other content<div>
        </section>
        <section>
            <div>Some more content<div>
            <div>Some other content<div>
        </section>
        <footer>
            <div>Logo</div>
            <div>Links</div>
        </footer>
    <body>
</html>
```
